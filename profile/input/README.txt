AutoDoc Watch Mode - Profile Input Directory
=============================================

Drop your files here to automatically generate documents.

Instructions:
1. Place a text file (.txt) and a template file (.docx, .pptx, or .xlsx) 
   with matching names into this directory.

   Example:
   - report.txt
   - report.docx

2. The watch mode will automatically detect the files and generate the 
   output document in the profile/output/ directory.

3. After processing, the input files will be moved to the 'processed/' 
   subdirectory.

For more information, see WATCH_MODE.md in the root directory.
